# 🎨 Colorfly Studio

## 📌 Descripción del proyecto
Consiste en una herramienta web simple que permite generar paletas de colores de manera rápida e intuitiva.  
Está diseñada para una agencia de branding que trabaja con más de 300 clientes en distintas ciudades, con el objetivo de facilitar la creación de propuestas visuales iniciales.

---

## 🎯 Objetivos
Evaluar si una herramienta web sencilla para generar paletas de colores mejora la velocidad y consistencia en la creación de propuestas visuales.

---

## ⚙️ Funcionalidades incluidas
- Generación automática de paletas de colores  
- Selección de cantidad de colores (6, 8 o 9)  
- Elección del formato de color (HEX o HSL)  
- Visualización de colores en pantalla  
- Posibilidad de regenerar combinaciones  
- Copiar códigos de color  
- Bloquear colores para conservarlos  
- Agregar más colores a la paleta existente  

---

## 🚫 Funcionalidades NO incluidas
- Guardado de paletas  
- Sistema de usuarios  
- Exportación avanzada (PDF, imágenes)  
- Personalización avanzada de reglas de color  

---

## 🛠️ Tecnologías utilizadas
- HTML  
- CSS  
- JavaScript  

---

## ▶️ Cómo usar la aplicación
1. Abrir la aplicación en el navegador  
2. Elegir el formato de color (HEX o HSL)  
3. Seleccionar la cantidad de colores  
4. Hacer clic en **"Generar paleta"**  
5. Hacer clic sobre un color para copiar su código  
6. (Opcional) Bloquear colores para mantenerlos  
7. Volver a generar la paleta las veces que desees  

---

## 🧠 Decisiones tomadas
Se priorizó la simplicidad y rapidez de uso para permitir al equipo creativo generar ideas espontáneas.  
La aplicación es adaptable a distintos dispositivos (móvil y escritorio) y busca ser intuitiva.
